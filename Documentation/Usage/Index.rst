=====
Usage
=====

For Administrators
==================

1. Go to the **List** module at root level (pid=0).
2. Create a new **Backend Theme** record.
3. Set a title and choose a primary color.
4. Optionally configure secondary color and dark mode overrides.
5. Check "Default Theme" to apply it to all users by default.

For Users
=========

1. Open **User Settings** (click your avatar in the top-right).
2. Under **Appearance**, find the **Custom Backend Theme** dropdown.
3. Select a theme from the list or choose "Standard" to use the
   default or TYPO3 built-in theme.
